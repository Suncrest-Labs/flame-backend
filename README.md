# Flame Backend

Flame Backend is a Go-based backend service powering a Web3 decentralized SocialFi application - Flame. It provides APIs and services to support decentralized social interactions and financial functionalities on Flame.

## Features

- **Decentralized Architecture**: Built to support Web3 principles.
- **SocialFi Integration**: Combines social networking with financial incentives.
- **Scalable Backend**: Designed for high performance and scalability using Go.
- **Secure and Reliable**: Implements best practices for security and reliability.

## API Documentation

API documentation is available in the `docs/` folder or can be accessed via the Swagger UI at `/swagger` when the server is running.
